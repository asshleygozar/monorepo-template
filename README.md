# Monorepo Template
A monorepo template for building full stack web application using next.js and express.js with monorepo manager, turborepo

## Technologies

- **Frontend**
    - Next.js v16.0.10
    - TypeScript v5
    - Tailwind v4
    - React.js v19.2.1

- **Backend**
    - Node.js v25.0.0
    - Express.js v5.1.0
    - Prisma v7.0.1
    - TypeScript v5.9.3

- **DevOps**
    - Turborepo v2.6.3
